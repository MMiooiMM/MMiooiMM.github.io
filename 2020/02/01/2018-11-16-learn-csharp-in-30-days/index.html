.net 後端起手式 30天認識 C#

Table of Contents

這陣子在 IT 鐵人賽撰寫了一系列的 C# 文章，一下子把今年預計的目標都達成了。本想說建立部落格後，一年內發個 20 篇文章，給自己一個期望。但參加鐵人賽完成後就相當於發了 30 篇文章，雖然觀看的人數沒有很多，也沒有任何人參與討論，一個人邊緣的寫完了。

以第三十天的總結來說明一下內容。

本系列從最基本的環境安裝與建立專案，到主要構成內容的型別與變數以及運算子，再到控制程式流程的陳述式。

增加程式碼可讀性與維護性，了解方法與物件的建立與使用。方法可以說是整合單一運算邏輯，而類別則是整合一個物件邏輯。類別本身也能重複使用，藉由使用繼承或者介面。

認識方法參數與型別參數，當作參數傳遞的方法為委派，當作參數傳遞的型別為泛型，也介紹了官方的泛型集合類別。

認識查詢運算式，先由迭代器來了解集合操作時的遍歷方式後，使用 LINQ 來操作集合查詢。

深入了解類型，查看類型資訊的類型 Type，並透過反射來查看類型內容，或對該類型存取屬性或使用方法。透過反射給予類型額外內容為特性。

了解往後的閱讀，知道 System 命名空間的更多內容觀看，以及更多更多的命名空間(NuGet)觀看。

這邊就不想一篇一篇的轉貼了，有興趣的朋友可以前往 .net 後端起手式 30天認識 C# 來觀看，想與我討論的在哪邊留言都可以唷。
